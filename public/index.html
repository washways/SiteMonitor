Welcome to the WASH Dashboard

Water Access & Sanitation Monitoring

This dashboard provides real-time monitoring of solar-powered and hand-pump waterpoints across Malawi.

📊
Flow Reports Track water flow totals, trends, and site status over configurable time periods
🗺
Interactive Map Visualise all waterpoints on a live map with colour-coded freshness indicators
📈
Site Detail Charts Dual-axis charts showing flow rates and water levels for each pump
⚙️
Two Data Sources Combines DCP Water API and SonSetLink API into a single unified view

Click ⚙ Settings in the header to enter your API credentials and start loading data.
